#### 说明书



![](/assets/0d4a5fd2929b/1*kuAtQ-dLWNNAjm37DuTWwA.png)



基本上跟所有机械键盘差不多，反正就灯光调整跟连线方式设定。



#### 启动流程



- 接线插到电脑上


- 长按 `FN + ESC` 3 秒以上键盘开机 (Tab 底光会闪)


- 切换成 macOS 键位配置： `FN + Tab` **(macOS 务必设定，不然 CMD 会变要按 CTRL)**


- 设定背光： `FN + [` / `FN + ]` / `FN + Shift` / `FN + Enter`
