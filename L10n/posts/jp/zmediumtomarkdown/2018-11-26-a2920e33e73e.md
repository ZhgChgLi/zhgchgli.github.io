### 入手から始めて…

#### 個人背景

まずは自分のApple製品使用歴について説明します。私は熱心なAppleファンではありません。初めて触れたのは2015年、アルバイトの給料で買ったiPhone 6でした。その後、仕事の都合で昨年までMacOSのパソコン（Mac Mini）を使い始め、今年になって自分のMacBook Proを購入し、iPhone 8に機種変更しました。私がAppleエコシステムに入った理由は主に以下の通りです：

1. 仕事のため（iOSアプリ開発には必ずMacOSデバイスが必要）

2. 作業効率（安定性やアプリの切り替え、操作体験が向上し、エコシステム内のiPhoneやMacOSとの連携、データ同期により多くの場面で複雑さを解消）

3. バッテリー持続時間、携帯性、Retinaディスプレイ

[2019–05–02更新]：Apple製品のラインナップに新たに加わったのは、[AirPods 2（開封と使用感はこちら）](../33afa0ae557d/)です。

#### なぜApple Watchを買いたかったのか？

1. 運動状況や心拍数の記録

2. ランニング中に携帯電話を持ちたくない

3. 携帯電話の使用時間を減らしたいが、重要な情報は見逃したくない

4. 大きな荷物があるときでも、スマホを取り出さずにApple Payを使いたい

5. MacBookの自動ロック解除に近づく（私のMacBook ProはTouch Bar非搭載モデルで、パスワード入力がとても疲れる）

6. 自転車でナビを見る

7. かっこいい！使ったことはないけど、試しに買ってみたい。

8. WatchOSアプリを作りたい

### 選び始め…

以上の要素を総合して、適したApple Watchを選び始めました。バンドの素材を除くと、本体には3つのバージョンがあります：

1. アルミニウムケース＋傷つきやすいガラス面＋GPS = 12,900円（40mm） / 13,900円（44mm）

2. アルミニウムケース＋傷つきやすいガラス面＋GPS＋モバイルネットワーク = 16,500円（40mm） / 17,500円（44mm）

3. ステンレススチールケース＋サファイアガラス＋GPS＋モバイル通信 = 22,900円（40mm） / 24,900円（44mm）

私自身は **2. アルミニウムケース＋傷がつきやすいガラス表面＋GPS＋モバイル通信 44mm** を購入しました。

#### **ケース素材**

アルミニウムケース＋傷がつきやすいガラス表面 と ステンレススチールケース＋サファイアガラス の2種類があります。予算に余裕がある方は後者をおすすめしますが、私は予算の関係で前者を選びました。なぜステンレススチールケース＋サファイアガラスのモデルを選ぶべきなのでしょうか？

1. 本体はやや重い（運動時に感じるかもしれません）が、日常生活ではファッションに合わせやすいです。レザーやメタルのバンドは、ケースが未加工の本体とビジネスウェアにマッチし、統一感のある品の良さを演出します。カジュアルや運動時にはスポーツバンドに替えても優雅さを失わず、アクティブにもリラックスにも対応できます。

2. サファイアガラスは非常に硬いため、画面の傷を気にする必要がありません（個人的な経験：前に使っていたiPhone 6は裸で1年以上使用しました。特に傷つけることなく、普段はポケットや机の上に置いていましたが、画面はかなり傷だらけになりました。しかし、レンズ部分はサファイアガラスなので新品同様のままでした）

#### モバイル通信対応モデルを買うべき？

この部分については保留的な態度です。私はモバイル通信モデルを購入しましたが、今後ランニングや運動時に携帯電話を持ち歩く必要がなくなるためです。また、2〜3年は使う予定で将来どうなるか分からないので先にアップグレードしました。しかし、予算が限られていて外出時に携帯電話を持たないことがないなら、WiFiモデルだけで十分です（価格差は$3600です）。  
以下の点を考慮してください：

1. 現在Spotifyはオフライン再生に対応していないため、運動中に音楽を聴くにはスマホが必要です（2018/11/21）  
   p.s. Apple MusicやKKBOXはオフライン再生に対応しているので、この問題はありません。

2. Apple Watchのアプリは多くなく、できることは電話をかける／メッセージに返信する／Lineに返信する／Fb Messengerに返信する／Apple Payだけです。  
   * Apple Payはセルラーモデルでなくてもオフラインで使用可能です。

3. モバイルネットワークの利用には別途申し込みが必要で、毎月199元の通信料金がかかります（中華電信の場合、2018年12月31日までの申し込みで割引価格149元）。データ通信量は元のスマホのプランを共有します。

4. モバイル通信の仕組みは、時計がデータを携帯電話に送信し、携帯電話がそれを通信キャリアに送る形です。したがって、**携帯電話も電源が入っている状態でないと時計は使えません。**  
   ***つまり、携帯電話の電池が切れて電源がオフだと、たとえモバイル通信契約があっても時計は使えません。***

### はじめに使い始める

ペアリングや基本設定についてはここでは詳しく説明しません。上記の開封レビューを参照してください。ここでは、すでに設定が完了しApple Watchの使用を開始した前提とします。

![ボタンの画像 — [Apple公式サポート](https://support.apple.com/zh-tw/HT205552){:target="_blank"}](/assets/a2920e33e73e/1*qNlLQb-sqqPPimwF5b1Wvw.png)

ボタンの画像を添付 — [Apple公式サポート](https://support.apple.com/zh-tw/HT205552){:target="_blank"}

了解しました。ご指示の用語は以下のように翻訳します。

- 「Digital Crown」→「デジタルクラウン」
- 「Side Button」→「サイドボタン」

以降の翻訳でこれらの用語を使用します。

**ボタン操作部分：**

1. デジタルクラウンを一度押すと、ホーム画面と文字盤を切り替えます。

2. デジタルクラウンをダブルクリックすると、最近使用したアプリに切り替わります。

3. 側面のボタンを押すとDock（マルチタスク画面）が表示されます。最近使ったアプリか、お気に入りのアプリを表示するように設定できます（「iPhone」の「Watch」アプリを開き、「マイウォッチ」タブ→Dock→Dockの並び順を設定）。

4. 側面のボタンをダブルクリックするとApple Payが呼び出され、そのまま支払いができます。  
   p.s Apple Payのデフォルトカードを変更するには、「iPhone」の「Watch」アプリを開き、「マイウォッチ」タブ → 「ウォレットとApple Pay」 → 「トランザクションのデフォルト」 → 「デフォルトカード」 → 設定したいカードを選択してください。  
   * 順序の変更はできず、特定のカードをデフォルトとして一番目に設定するのみです。

5. 側面ボタンを長押しすると、システムメニューの「電源オフ」や「再起動」、医療IDの表示、SOS緊急電話の発信ができます。

#### Apple Watch のスクリーンショット機能

とても重要なので最初に書きます。Apple Watchのスクリーンショットの撮り方：
「iPhone」の「Watch」アプリを開く -> 「マイウォッチ」タブ -> 「一般」へ進む -> 「スクリーンショットを有効にする」をオンにする

![](/assets/a2920e33e73e/1*Za5IVCeJy_kEwoprlvgWkA.png)

Apple Watchでデジタルクラウンとサイドボタンを同時に押すと、画面に光のスワイプ効果が表示され、スクリーンショットが完了します。その後、iPhoneを開くとスクリーンショットの画像を見ることができます！

#### iPhoneのAPPの通知をApple Watchに送信する方法

時計はデフォルトでiPhoneのアプリ通知設定を反映しますが、特定のアプリの通知を時計に送らないように個別にオフにすることもできます（iPhoneの「Watch」アプリを開き -> 「マイウォッチ」ページ -> 「通知」 -> 一番下までスクロールして各アプリの設定を調整）。

- もしアプリがこのリストに表示されていない場合、そのアプリはもともとiPhoneで通知がオンになっていません（「iPhone」の「設定」->「通知」->該当アプリの通知をオンにしてください）。

- なぜ通知によって音や振動があるものとないものがあるのですか？  
  この設定はiPhoneのアプリ通知設定に依存しており、アプリの「通知」で「サウンド」がオンになっていると音や振動が発生します。

- ほとんどのAPP通知は閲覧のみ対応で、一部（例えばLineの通知）は時計上で返信操作が可能です。

- **スマホ未使用状態＋時計装着中のみ、時計に新しい通知が表示されます／スマホ側は鳴りませんが通知センターには表示されます；スマホと時計が同時に鳴るのを防ぎます**

#### Apple Watch対応のアプリの場合

- デフォルトでは、アプリをインストールすると、そのアプリがApple Watch対応の場合はApple Watchにも自動でインストールされます（「iPhone」の「Watch」アプリ -> 「マイウォッチ」タブ -> 「一般」 -> 「自動アプリインストール」をオフにできます）。

- Apple Watchのアプリだけをインストールできますか？  
  いいえ、現時点ではApple Watchのアプリを単独でインストールすることはできません。必ずiPhone側に対応するアプリが存在します。

- Apple Watch版のアプリをインストールしたくない場合  
  「iPhone」の「Watch」アプリ -> 「マイウォッチ」タブ -> 下にスクロールして「Apple Watchにインストール済み」セクションを開く -> 「Apple WatchにAppを表示」をオフにする

- **APPが「複雑な機能」をサポートしているということは、文字盤のウィジェットをサポートしているという意味です**

#### 各モード

「サイレントモード」- すべての通知が無音、振動なし、画面点灯なしで通知センターにのみ表示されます

「シアターモード」- 手首を上げても画面が点灯せず、画面をタップすると点灯します

「水中ロック」- 画面のタッチをロックし、デジタルクラウンを回して解除します。解除後、スピーカーが自動で音を出して水を排出します。

「機内モード」- すべての外部接続をオフにする

「省電モード」- 本当に電池が長持ちします！デジタルクラウンを押すと時間が表示されるだけで、他の機能はすべてオフになり、ほぼシャットダウン状態と同じです。省電モードを解除するには、サイドボタンを長押しします（電源オンと同じ操作）。

以上のすべてのモード（アラーム、カウントダウン機能）は、省電力モードでも強制的に起動して正常に鳴ります。

#### 手首を上げて直接Siriを呼び出す

手首を上げて画面が点灯すると、直接Siriに話しかけることができます！「Hey! Siri」と言う必要はありません（例：手首を上げてから直接 **「明日の天気」** と話す）。
スマホが離れている時でもSiriを使えます（例：洗濯物を干している時）。

#### AQIの空気質が表示されない？

内蔵のAQIは台湾地域をサポートしていないようです。「App Store」で「在意空氣」を検索してダウンロード・インストールし、起動した後、文字盤の複雑機能の設定画面で「在意空氣」を選択してください。

#### Apple WatchでMacのロックを解除する

1. iPhone、Apple Watch、Macが同じApple IDでサインインしていることを確認してください。

2. Appleアカウントで[二要素認証](https://support.apple.com/zh-tw/HT205075){:target="_blank"}が有効になっていることを確認してください。

3. システムがあなたのAppleアカウントにApple Watchデバイスを検出すると、「システム環境設定」->「セキュリティとプライバシー」->「一般」に「Apple WatchでMacのロックを解除を許可」という項目が追加され、チェックを入れるだけで設定完了です。

もし有効化に失敗し続ける場合は、まずAppleアカウントで二要素認証（[二段階認証](https://support.apple.com/zh-tw/HT207198){:target="_blank"} ではありません）が有効になっているか確認し、パソコンの再起動を試してください！

p.s 私の会社のMac Miniはずっと起動できず、再起動すると正常に動作しました。

#### 写真が開くと真っ白になる？

デフォルトでiPhoneのお気に入り項目が表示されます。iPhoneの「写真」アプリを開き、Apple Watchに送信したい写真を「ハートマーク」をタップすると表示されます。

#### アクティビティ記録とフィットネストレーニング

活動記録は毎日3つのリングと3つの目標があります：

1. 立つ（青）：毎時間1分間立つと1回達成とみなされます

2. 運動（緑）：早歩き以上の強度の活動時間のみがカウントされます

3.アクティビティ（赤）：燃焼した動的カロリー数で、動いていると増加します

詳細はiPhoneの「ヘルスケア」アプリでご確認ください。

毎日の達成記録が通知され、さらにApple Watchの「アクティビティ」アプリを強く押して活動目標を調整できます（デフォルトでは1日360キロカロリーが目標です）。

体力トレーニングのランニングはNike Run Clubを使っており、内蔵のものは使っていません。先週、自転車に乗った時に内蔵の体力トレーニング「屋外サイクリング」を試してみました。高度、距離、時間、ルート、心拍数を記録してくれて、とても良かったです！

#### 地図機能？

現在はApple Mapのみ対応しており、Google Mapは未対応です。「マップ」を開いて検索するか、個人情報設定の会社や自宅の住所（連絡先情報->私の名刺から）や連絡先、または目標を手動で入力します。ナビゲーションを開始すると、各曲がり角がカードとして表示され、走行に合わせて自動でページが切り替わります。カードを回転させて確認でき、タップすると地図の詳細を表示します。残り距離が40メートルになると振動で通知し、強く押すとナビゲーションを終了できます。

#### メッセージ入力で手書きの中国語が認識できず、音声も中国語を聞き取れない

これはバグだと思う…

![メッセージ内で「マイク」または「手書き」アイコンを長押ししてメニューを呼び出し > 「言語を選択」->「中国語」](/assets/a2920e33e73e/1*KjBwFaHI3Aw894vw8RN3kw.jpeg)

これで音声入力は中国語（普通話）のみ認識するようになり、スマートフォンの部分も同様に影響を受けます。

#### バッテリーの持ちはどうですか？

私の装着時間は起床から入浴前までで、寝るときは付けません（壁に寄りかかって無意識に壁を叩くのを避けるため）。入浴前に外して充電しています。

- 夜12時に満充電で外し、そのまま放置すると、翌朝8時には約95%残っています。

- 夜12時にフル充電して外し、**機内モードに切り替えた**まま放置すると、翌朝8時には約98%残っています。

一日中で約15時間装着し、特にずっと操作しなければ残りのバッテリーは約65％で、かなり持ちます。無理すれば2日に1回の充電も可能です。

*初回の充電は時間がかかる場合があります  
*最初の数日はバッテリー性能が十分に発揮されず、消耗が激しいことがあります

### まとめと1週間の使用感想

装着してからもうすぐ2週間になります。最初は新鮮な気持ちでしたが、今では生活に自然と溶け込んでいます。これまで生活に役立ったことは、MACのロック解除で長いパスワードを入力しなくて済むこと（会社の規定で席を離れるとログアウトが必要）、リアルタイムの天気確認、ナビの確認、アプリの通知、心拍数のチェックで健康を気にかけるくらいです。対応しているアプリや機能は非常に少ないです。

携帯電話の使用時間は減りましたか？特に感じません。通知を受け取っても、私はまだスマホで返信する習慣があります。時計で返信するには音声入力が必要ですが、人前では使いにくいですし、手書き入力は非常に遅いです。また、多くのアプリがApple Watchに対応していません。

動くと12,900円から始まるのは本当に価値があるのでしょうか？1万円を超える時計にはもっと良い選択肢がたくさんありますが、Apple製品と連携できるのはこれだけです。もし単に高級時計を買いたいだけなら、Apple Watchを使う必要はありません。日常の細かい問題を解決できる時計が欲しいなら検討してもいいでしょう。贅沢品としても日常の問題も解決したいなら、ステンレススチールやHermès版も考慮してください！

購入してから一度は返品しようかと思ったことがあります。17,500ドルあれば色々なことができるのに、腕時計にそんなにお金を使うのはあまり価値がないように感じました。しかし、確かに日常生活には役立っています。この役立ち度が17,500ドルの価値があるかと言われると、今のところはないと思います。Apple Watchのアプリエコシステムがもっと充実してからまた評価したいですね。現状は贅沢品です（笑）。気持ちよさやトレンド、衝動で買いました。

その他の点は皆さん自身で体験してみてくださいね。
