### ZReviewsBot — Slack アプリレビュー通知ボット

無料オープンソースの iOS & Android アプリ最新レビュー追跡 Slack Bot

### TL;DR [2022/08/10] アップデート:

現在は新しい [App Store Connect API](../f1365e51902c/) を使用して App Reviews Bot を再設計し、「[ZReviewTender — 無料オープンソースの App Reviews 監視ボット](../e36e48bb9265/)」としてリニューアルしました。

[ZReviewsBot](https://github.com/ZhgChgLi/ZReviewsBot){:target="_blank"} は無料のオープンソースプロジェクトで、App チームが App Store (iOS) と Google Play (Android) の最新レビューを自動で追跡し、指定した Slack チャンネルに送信してリアルタイムでアプリの状況を把握できるよう支援します。

- ✅ より新しく信頼性の高い API エンドポイントを使って iOS アプリのレビューを追跡 ( [技術詳細](../cb0c68c33994/) )

- ✅ iOS と Android の両プラットフォームのレビュー追跡に対応

- ✅ キーワード通知スキップ機能対応（スパム広告対策）

- ✅ カスタマイズ設定が自由自在

- ✅ GitHub Action を使ったスケジュール自動化ボットのデプロイに対応

### [2022/07/20 更新]

[App Store Connect API は現在、カスタマーレビューの読み取りと管理をサポートしています](../f1365e51902c/)。このボットは今後のアップデートで実装され、Fastlane — Spaceship を使ってバックエンドからレビューを取得する方法に代わります。

### 起源

前回の「[AppStore APP’s Reviews Slack Bot 那些事](../cb0c68c33994/)」の研究に続き、新しい iOS 評価取得ツールを完成させました。同じ問題を抱える方々のために、Side Projectとしてオープンソースで公開するのが良さそうだと思いました。
