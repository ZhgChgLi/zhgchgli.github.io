### Slack と ChatGPT の統合

ChatGPT OpenAI API を使った Slack アプリを自作する方法（Google Cloud Functions と Python）

#### 背景

最近、チーム内でGenerative AIを活用して業務効率を向上させる取り組みを推進しています。まずはAIアシスタント（ChatGPT機能）を導入し、日常のデータ検索や煩雑な資料整理、手作業によるデータ処理の時間を削減して業務効率を上げることを目指しています。エンジニア、デザイナー、PM、マーケティング担当者など、誰でも自由に使える環境にしたいと考えています。

最も簡単な方法は、ChatGPT Teamプランを直接購入することで、1席あたり年間25ドルです。しかし、利用頻度や量がまだ不明であり、外部との連携やより多くの協力・開発プロセスとの統合を希望しているため、OpenAI APIを利用し、他のサービスを通じてチームメンバーが使いやすいように統合する方法を採用しました。

OpenAI API Key は [こちらのページ](https://platform.openai.com/api-keys){:target="_blank"} から生成できます。Key 自体には対応するモデルバージョンの区別はなく、使用時に利用するモデルバージョンを指定して、それに応じたトークン料金が発生します。

> 私たちは、自分でOpenAI APIキーを設定し、そのキーを使ってChatGPTのように利用できるサービスが必要です。

> Chrome拡張機能やSlackアプリでも、自分でOpenAI APIキーを設定できるサービスはほとんど見つかりません。多くのサービスは独自のサブスクリプションを販売しており、ユーザーがAPIキーを自由に設定できると収益にならないため、慈善事業のようになってしまいます。

### 自分で実装する ChatGPT OpenAI API for Slack App

以前にいくつかSlackアプリの開発経験があり、自分で作ることに決めました。

> **⚠️免責事項⚠️**

> 本文ではOpenAI APIの連携例として、Slack Appの作成方法とGoogle Cloud Functionsを使った迅速な実装方法を紹介します。Slack Appはさまざまな用途に活用できるので、自由に応用してください。

> ⚠️⚠️ Google Cloud Functions、Function as a Service（FaaS）の利点は、手軽で迅速に使え、無料枠があり、コードを書けばすぐにデプロイして実行でき、自動でスケールすることです。欠点はサービス環境がGCPに管理されているため、長時間呼び出されないと休止状態になり、その後の呼び出しが[Cold Start](https://cloud.google.com/functions/docs/configuring/recommender?hl=en){:target="_blank"}（コールドスタート）となり、応答時間が長くなることです。また、複数のサービスを連携させるのも難しいです。

> より完全で使用量が多い場合は、やはり自分でVM（App Engine）を立ててサーバーを運用することをお勧めします。

Runtime: Python 3.8 またはそれ以上

`main.py` ：

```python
import functions_framework

@functions_framework.http
def hello_http(request):
    request_json = request.get_json(silent=True)
    request_args = request.args
    request_headers = request.headers

    # 実行時のログを簡単に print で記録可能。ログは Logs で確認できる
    # 詳細なログレベルについては以下を参照：https://cloud.google.com/logging/docs/reference/libraries
    print(request_json)

    # 制限のある FAAS（Cloud Functions）は、長時間呼び出されないとコールドスタートになるため、
    # Slack の規定する3秒以内に応答できないことがある
    # さらに OpenAI API のリクエストから応答まで時間がかかる（応答の長さによっては約1分かかることも）
    # Slack は制限時間内に応答がないとリクエストが失われたと見なし、再度呼び出しを行う
    # これにより重複リクエストや応答が発生するため、レスポンスヘッダーに X-Slack-No-Retry: 1 を設定し、
    # 期限内に応答がなくても Slack にリトライ不要と通知する
    headers = {'X-Slack-No-Retry':1}

    # Slack のリトライリクエストの場合は無視する
    if request_headers and 'X-Slack-Retry-Num' in request_headers:
        return ('OK!', 200, headers)

    # Slack App イベントサブスクリプションの検証
    # https://api.slack.com/events/url_verification
    if request_json and 'type' in request_json and request_json['type'] == 'url_verification':
        challenge = ""
        if 'challenge' in request_json:
            challenge = request_json['challenge']
        return (challenge, 200, headers)

    return ("Access Denied!", 400, headers)
```

以上のJsonペイロードを基に、SlackメッセージをOpenAI APIへ送り、再びSlackメッセージとして返信する連携を完成させることができます：

**Cloud Functions `main.py` ：**

```python
import functions_framework
import requests
import asyncio
import json
import time
from openai import AsyncOpenAI

OPENAI_API_KEY = "OPENAI API KEY"
SLACK_BOT_TOKEN = "Bot User OAuth Token"

# 使用する OPENAI API モデル
# https://platform.openai.com/docs/models
OPENAI_MODEL = "gpt-4-1106-preview"

@functions_framework.http
def hello_http(request):
    request_json = request.get_json(silent=True)
    request_args = request.args
    request_headers = request.headers

    # 実行時のログを簡単に print で記録でき、Logs で確認可能
    # 詳細なログレベルの使用は参考：https://cloud.google.com/logging/docs/reference/libraries
    print(request_json)

    # FAAS(Cloud Functions)の制約で、長時間呼び出しがないとコールドスタートが発生し、
    # Slackの3秒以内の応答制限を超える可能性あり
    # また、OpenAI APIの応答に時間がかかる場合があり（応答長により最大1分近く）
    # Slackは期限内に応答がないとリクエストを失ったと判断し再度呼び出す
    # これにより重複リクエスト・応答の問題が起きるため、レスポンスヘッダーに
    # X-Slack-No-Retry: 1 を設定してSlackに期限外でも再試行不要を伝える
    headers = {'X-Slack-No-Retry':1}

    # Slackのリトライリクエストの場合は無視
    if request_headers and 'X-Slack-Retry-Num' in request_headers:
        return ('OK!', 200, headers)

    # Slack App Event Subscriptions の検証
    # https://api.slack.com/events/url_verification
    if request_json and 'type' in request_json and request_json['type'] == 'url_verification':
        challenge = ""
        if 'challenge' in request_json:
            challenge = request_json['challenge']
        return (challenge, 200, headers)

    # イベントサブスクリプションのイベント処理...
    if request_json and 'event' in request_json and 'type' in request_json['event']:
        # イベント発生元が自分のSlack Appの場合は無視し無限ループ防止
        if 'api_app_id' in request_json and 'app_id' in request_json['event'] and request_json['api_app_id'] == request_json['event']['app_id']:
            return ('OK!', 200, headers)

        # イベント名（例：message、app_mentionなど）
        eventType = request_json['event']['type']

        # サブタイプ（例：message_changed、message_deleted）
        # 新規メッセージにはサブタイプなし
        eventSubType = None
        if 'subtype' in request_json['event']:
            eventSubType = request_json['event']['subtype']
        
        if eventType == 'message':
            # サブタイプがあれば編集、削除、返信などなので無視
            if eventSubType is not None:
                return ("OK!", 200, headers)
               
            # イベントメッセージの送信者
            eventUser = request_json['event']['user']
            # イベントメッセージのチャンネル
```

```python
    if request_json and 'event' in request_json and 'type' in request_json['event']:
        apiAppID = None
        if 'api_app_id' in request_json:
            apiAppID = request_json['api_app_id']
        # イベントの発生元がAppでかつApp IDがSlack App IDと一致する場合は
        # 自身のSlack Appがトリガーしたイベントなので無視（無限ループ防止）
        if 'app_id' in request_json['event'] and apiAppID == request_json['event']['app_id']:
            return ('OK!', 200, headers)

        # イベント名（例：message（メッセージ関連）、app_mention（メンション）など）
        eventType = request_json['event']['type']

        # SubType（例：message_changed（メッセージ編集）、message_deleted（メッセージ削除）など）
        # 新規メッセージにはSub Typeなし
        eventSubType = None
        if 'subtype' in request_json['event']:
            eventSubType = request_json['event']['subtype']
        
        if eventType == 'message':
            # Sub Typeがある場合は編集、削除、返信などなので無視
            if eventSubType is not None:
                return ("OK!", 200, headers)
               
            # イベントのメッセージ送信者
            eventUser = request_json['event']['user']
            # イベントのメッセージが属するチャンネル
            eventChannel = request_json['event']['channel']
            # イベントのメッセージ内容
            eventText = request_json['event']['text']
            # イベントのメッセージTS（メッセージID）
            eventTS = request_json['event']['event_ts']
                
            # イベントメッセージのスレッド親メッセージTS（メッセージID）
            # スレッド内の新規メッセージのみ存在する
            eventThreadTS = None
            if 'thread_ts' in request_json['event']:
                eventThreadTS = request_json['event']['thread_ts']
                
            openAIRequest(apiAppID, eventChannel, eventTS, eventThreadTS, eventText)
            return ("OK!", 200, headers)

    
    # Shortcut（メッセージ）の処理
    if payload and 'type' in payload:
        payloadType = payload['type']

        # メッセージShortcutの場合
        if payloadType == 'message_action':
            callbackID = None
            channel = None
            ts = None
            text = None
            triggerID = None

            if 'callback_id' in payload:
                callbackID = payload['callback_id']
            if 'channel' in payload:
                channel = payload['channel']['id']
            if 'message' in payload:
                ts = payload['message']['ts']
                text = payload['message']['text']
            if 'trigger_id' in payload:
                triggerID = payload['trigger_id']
            
            if channel is not None and ts is not None and text is not None:
                # OpenAI APIの応答生成停止Shortcutの場合
                if callbackID == "abort_openai_api":
                    slackUpdateMessage(channel, ts, {"event_type": "aborted", "event_payload": { }}, text)
                    if triggerID is not None:
                        slackOpenModal(triggerID, callbackID, "OpenAI APIの応答生成停止に成功しました！")
                        return ("OK!", 200, headers)


    return ("Access Denied!", 400, headers)

async def openAIRequestAsync(eventChannel, eventTS, messages):
    client = AsyncOpenAI(
      api_key=OPENAI_API_KEY,
    )

    # ストリームレスポンス（分割応答）
    stream = await client.chat.completions.create(
      model=OPENAI_MODEL,
      messages=messages,
      stream=True,
    )
    
    result = ""

    try:
        debounceSlackUpdateTime = None
        async for chunk in stream:
            result += chunk.choices[0].delta.content or ""
            
            # 0.8秒ごとにメッセージを更新し、Slack Update APIの呼び出し過多を防止
            if debounceSlackUpdateTime is None or time.time() - debounceSlackUpdateTime >= 0.8:
                response = slackUpdateMessage(eventChannel, eventTS, None, result+"...")
                debounceSlackUpdateTime = time.time()

                # メッセージにmetadataがあり、metadataのevent_typeがabortedならユーザーが応答を中断した
                if response and 'ok' in response and response['ok'] == True and 'message' in response and 'metadata' in response['message'] and 'event_type' in response['message']['metadata'] and response['message']['metadata']['event_type'] == "aborted":
                    break
                    result += "...*[中断されました]*"
                # メッセージが削除されている場合
                elif response and 'ok' in response and response['ok'] == False and 'error' in response and response['error'] == "message_not_found" :
                    break
                
        await stream.close()
                
    except Exception as e:
        print(e)
        result += "...*[エラーが発生しました]*"

    slackUpdateMessage(eventChannel, eventTS, None, result)

def slackRequest(endpoint, method, payload):
    url = "https://slack.com/api"+endpoint

    headers = {
        "Authorization": f"Bearer {SLACK_BOT_TOKEN}",
        "Content-Type": "application/json",
    }

    response = None
    if method == "POST":
        response = requests.post(url, headers=headers, data=json.dumps(payload))
    elif method == "GET":
        response = requests.post(url, headers=headers)

    if response and response.status_code == 200:
        result = response.json()
        return result

        return None
```

```python
import functions_framework
import requests
import re
import asyncio
import json
import time
from openai import AsyncOpenAI

OPENAI_API_KEY = "OPENAI API KEY"
SLACK_BOT_TOKEN = "Bot User OAuth Token"

# 独自定義のセキュリティトークン
# URLに ?token=SAFE_ACCESS_TOKEN パラメータが付いている場合のみリクエストを受け付ける  
SAFE_ACCESS_TOKEN = "nF4JwxfG9abqPZCJnBerwwhtodC28BuC"

# 使用する OPENAI API モデル
# https://platform.openai.com/docs/models
OPENAI_MODEL = "gpt-4-1106-preview"

@functions_framework.http
def hello_http(request):
    request_json = request.get_json(silent=True)
    request_args = request.args
    request_headers = request.headers

    # Shortcut のイベントは post payload フィールドから受け取る
    # https://api.slack.com/reference/interaction-payloads/shortcuts
    payload = request.form.get('payload')
    if payload is not None:
        payload = json.loads(payload)

    # 実行時のログは print で簡単に記録可能、Logs で確認できる
    # 詳細なログレベルは以下を参照：https://cloud.google.com/logging/docs/reference/libraries
    # print(payload)

    # FAAS(Cloud Functions)はしばらく呼ばれないとコールドスタート状態になり、Slackの3秒レスポンス制限内に応答できない可能性がある
    # さらにOpenAI APIの応答も時間がかかる（応答長によっては1分近くかかることも）
    # Slackは制限時間内に応答がないとリクエストを失ったと判断し再度呼び出すため
    # 重複リクエストや応答の問題が起きる
    # そこでレスポンスヘッダに X-Slack-No-Retry: 1 を設定し、制限時間を超えてもSlackに再試行不要と通知する    
    headers = {'X-Slack-No-Retry':1}

    # token パラメータの検証
    if not(request_args and 'token' in request_args and request_args['token'] == SAFE_ACCESS_TOKEN):
        return ('', 400, headers)

    # Slackのリトライリクエストは無視する
    if request_headers and 'X-Slack-Retry-Num' in request_headers:
        return ('OK!', 200, headers)

    # Slack Appのイベントサブスクリプション検証
    # https://api.slack.com/events/url_verification
    if request_json and 'type' in request_json and request_json['type'] == 'url_verification':
        challenge = ""
        if 'challenge' in request_json:
            challenge = request_json['challenge']
        return (challenge, 200, headers)

    # イベントサブスクリプションのイベント処理
    if request_json and 'event' in request_json and 'type' in request_json['event']:
        apiAppID = None
        if 'api_app_id' in request_json:
            apiAppID = request_json['api_app_id']
        # イベントの発信元が自分のSlack Appの場合、無限ループ防止のため無視する
        if 'app_id' in request_json['event'] and apiAppID == request_json['event']['app_id']:
            return ('OK!', 200, headers)

        # イベントタイプ例：message（メッセージ関連）、app_mention（メンションされた）など
        eventType = request_json['event']['type']

        # サブタイプ例：message_changed（メッセージ編集）、message_deleted（メッセージ削除）など
        # 新規メッセージはサブタイプなし
        eventSubType = None
        if 'subtype' in request_json['event']:
            eventSubType = request_json['event']['subtype']
        
        # メッセージ関連イベント
        if eventType == 'message':
            # サブタイプがあるものは編集、削除、返信などなので無視
            if eventSubType is not None:
                return ("OK!", 200, headers)
               
            # イベントメッセージの送信者
            eventUser = request_json['event']['user']
            # イベントメッセージのチャンネル
            eventChannel = request_json['event']['channel']
            # イベントメッセージの内容
            eventText = request_json['event']['text']
            # イベントメッセージのTS（メッセージID）
            eventTS = request_json['event']['event_ts']
                
            # イベントメッセージのスレッド親メッセージTS（メッセージID）
            # スレッド内の新規メッセージのみ存在
            eventThreadTS = None
            if 'thread_ts' in request_json['event']:
                eventThreadTS = request_json['event']['thread_ts']
                
            openAIRequest(apiAppID, eventChannel, eventTS, eventThreadTS, eventText)
            return ("OK!", 200, headers)
        
        # メンションイベント（@SlackApp こんにちは）
        if eventType == 'app_mention':
            # イベントメッセージの送信者
            eventUser = request_json['event']['user']
            # イベントメッセージのチャンネル
            eventChannel = request_json['event']['channel']
            # イベントメッセージの内容、先頭のメンション文字列 <@SLACKAPPID> を除去
            eventText = re.sub(r"<@\w+>\W*", "", request_json['event']['text'])
            # イベントメッセージのTS（メッセージID）
            eventTS = request_json['event']['event_ts']
                
            # イベントメッセージのスレッド親メッセージTS（メッセージID）
            # スレッド内の新規メッセージのみ存在
            eventThreadTS = None
            if 'thread_ts' in request_json['event']:
                eventThreadTS = request_json['event']['thread_ts']
                
            openAIRequest(apiAppID, eventChannel, eventTS, eventThreadTS, eventText)
            return ("OK!", 200, headers)

    
    # Shortcut（メッセージ）の処理
    if payload and 'type' in payload:
        payloadType = payload['type']

        # メッセージショートカットの場合
        if payloadType == 'message_action':
            callbackID = None
            channel = None
            ts = None
            text = None
            triggerID = None

            if 'callback_id' in payload:
                callbackID = payload['callback_id']
            if 'channel' in payload:
                channel = payload['channel']['id']
            if 'message' in payload:
                ts = payload['message']['ts']
                text = payload['message']['text']
            if 'trigger_id' in payload:
                triggerID = payload['trigger_id']
            
            if channel is not None and ts is not None and text is not None:
                # OpenAI APIの応答生成停止ショートカットの場合
                if callbackID == "abort_openai_api":
                    slackUpdateMessage(channel, ts, {"event_type": "aborted", "event_payload": { }}, text)
                    if triggerID is not None:
                        slackOpenModal(triggerID, callbackID, "OpenAI APIの応答生成停止に成功しました!")
                        return ("OK!", 200, headers)
                # メッセージ削除の場合
                if callbackID == "delete_message":
                    slackDeleteMessage(channel, ts)
                    if triggerID is not None:
                        slackOpenModal(triggerID, callbackID, "Slack Appのメッセージ削除に成功しました!")
                        return ("OK!", 200, headers)

    return ("Access Denied!", 400, headers)

async def openAIRequestAsync(eventChannel, eventTS, messages):
    client = AsyncOpenAI(
      api_key=OPENAI_API_KEY,
    )

    # ストリームレスポンス（分割応答）
    stream = await client.chat.completions.create(
      model=OPENAI_MODEL,
      messages=messages,
      stream=True,
    )
    
    result = ""

    try:
        debounceSlackUpdateTime = None
        async for chunk in stream:
            result += chunk.choices[0].delta.content or ""
            
            # 0.8秒ごとにメッセージを更新し、Slackの更新APIの呼び出し過多を防止し、Cloud Functionsの呼び出し回数節約
            if debounceSlackUpdateTime is None or time.time() - debounceSlackUpdateTime >= 0.8:
                response = slackUpdateMessage(eventChannel, eventTS, None, result+"...")
                debounceSlackUpdateTime = time.time()

                # メッセージに metadata があり、metadataの event_type が aborted の場合はユーザーが応答を中止したと判断
                if response and 'ok' in response and response['ok'] == True and 'message' in response and 'metadata' in response['message'] and 'event_type' in response['message']['metadata'] and response['message']['metadata']['event_type'] == "aborted":
                    break
                    result += "...*[中止済み]*"
                # メッセージが削除されている場合
                elif response and 'ok' in response and response['ok'] == False and 'error' in response and response['error'] == "message_not_found" :
                    break
                
        await stream.close()
                
    except Exception as e:
        print(e)
        result += "...*[エラー発生]*"

    slackUpdateMessage(eventChannel, eventTS, None, result)

def slackGetReplies(channel, ts):
    endpoint = "/conversations.replies?channel="+channel+"&ts="+ts
    response = slackRequest(endpoint, "GET", None)
    
    if response is not None and 'messages' in response:
        return response['messages']
    return None

def slackOpenModal(trigger_id, callback_id, text):
    slackRequest("/views.open", "POST", {
        "trigger_id": trigger_id,
        "view": {
            "type": "modal",
            "callback_id": callback_id,
            "title": {
                "type": "plain_text",
                "text": "通知"
            },
            "blocks": [
                {
                    "type": "section",
                    "text": {
                        "type": "mrkdwn",
                        "text": text
                    }
                }
            ]
        }
    })

def slackDeleteMessage(channel, ts):
    endpoint = "/chat.delete"
    payload = {
        "channel": channel,
        "ts": ts
    }
    
    response = slackRequest(endpoint, "POST", payload)
    return response

def slackUpdateMessage(channel, ts, metadata, text):
    endpoint = "/chat.update"
    payload = {
        "channel": channel,
        "ts": ts
    }
    if metadata is not None:
        payload['metadata'] = metadata
    
    payload['text'] = text
    
    response = slackRequest(endpoint, "POST", payload)
    return response

def slackRequestPostMessage(channel, target_ts, text):
    endpoint = "/chat.postMessage"
    payload = {
        "channel": channel,
        "text": text,
    }
    if target_ts is not None:
        payload['thread_ts'] = target_ts

    response = slackRequest(endpoint, "POST", payload)

    if response is not None and 'ts' in response:
        return response['ts']
    return None

def slackRequest(endpoint, method, payload):
    url = "https://slack.com/api"+endpoint

    headers = {
        "Authorization": f"Bearer {SLACK_BOT_TOKEN}",
        "Content-Type": "application/json",
    }

    response = None
    if method == "POST":
        response = requests.post(url, headers=headers, data=json.dumps(payload))
    elif method == "GET":
        response = requests.post(url, headers=headers)

    if response and response.status_code == 200:
        result = response.json()
        return result
    else:
        return None
```
