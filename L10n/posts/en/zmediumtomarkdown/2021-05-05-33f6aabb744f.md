### ZReviewsBot — Slack App Review Notification Bot

Free and Open Source iOS & Android App Latest Review Tracking Slack Bot

### TL;DR [2022/08/10] Update:

The App Reviews Bot has been redesigned using the brand-new [App Store Connect API](../f1365e51902c/) and relaunched as "[ZReviewTender — Free and Open Source App Reviews Monitoring Bot](../e36e48bb9265/)".

[ZReviewsBot](https://github.com/ZhgChgLi/ZReviewsBot){:target="_blank"} is a free, open-source project that helps your app team automatically track the latest reviews of your app on the App Store (iOS) and Google Play (Android) platforms, and send them to a designated Slack channel for real-time updates on your app’s status.

- ✅ Track iOS app reviews using updated and more reliable API endpoints ([Technical Details](../cb0c68c33994/))

- ✅ Supports dual-platform review tracking for iOS & Android

- ✅ Supports keyword notification skip feature (prevents spam and ad flooding)

- ✅ Customizable settings, as you wish

- ✅ Supports deploying scheduled automation bots using Github Action

### [2022/07/20 Update]

[App Store Connect API now supports reading and managing Customer Reviews](../f1365e51902c/). This bot will implement this in future updates to replace the current method of fetching reviews from the backend using Fastlane — Spaceship.

### Origin

Following the previous study "[AppStore APP’s Reviews Slack Bot 那些事](../cb0c68c33994/)," I completed a new iOS review fetching tool. I thought it would be suitable as a side project open source for others facing the same issue.
