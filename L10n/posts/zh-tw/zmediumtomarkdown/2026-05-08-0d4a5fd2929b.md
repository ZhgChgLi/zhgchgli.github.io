#### 啟動流程
- 接線插到電腦上
- 長按 `FN + ESC` 3 秒以上鍵盤開機 (Tab 底光會閃)
- 切換成 macOS 鍵位配置： `FN + Tab` **(macOS 務必設定，不然 CMD 會變要按 CTRL)**
- 設定背光： `FN + [` / `FN + ]` / `FN + Shift` / `FN + Enter`
